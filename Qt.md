# Install Qt6
1.Download Qt Creator
[Qt Creator](https://wiki.archlinux.org/title/Qt)
2. Install Qt
To check the Qt6.2.4 in Qt Creator and click the next button.
2.Install reference libs
```bash
sudo pacman -S glew
sudo pacman -S gxkb
sudo pacman -S libxkbcommon
sudo pacman -S mesa
sudo pacman -S libglvnd
sudo pacman -S vulkan-headers vulkan-icd-loader
```
